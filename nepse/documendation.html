NEPSE API Documentation

Welcome to the NEPSE (Nepal Stock Exchange) API documentation. This API provides access to various market data, indices, and trading information.

Market Data

  • NEPSE Subindices -> /
  • Securities Listing -> /security?nonDelisted=true
  • Market Status -> /nepse-data/market-open
  • Daily Trade Stat [price-volume]-> /securityDailyTradeStat/58
  • Corporate Diclosures -> /news/companies/disclosure

Graph Data

Index Graph

  • nepse -> /graph/index/58, POST
  • sensitive -> /graph/index/57, POST
  • float -> /graph/index/62, POST
  • sensitive_float -> /graph/index/63, POST

Sub Index Graph

  • banking -> /graph/index/51, POST
  • development_bank -> /graph/index/55, POST
  • finance -> /graph/index/60, POST
  • hotel_tourism -> /graph/index/52, POST
  • hydro -> /graph/index/54, POST
  • investment -> /graph/index/67, POST
  • life_insurance -> /graph/index/65, POST
  • manufacturing -> /graph/index/56, POST
  • microfinance -> /graph/index/64, POST
  • mutual_fund -> /graph/index/66, POST
  • non_life_insurance -> /graph/index/59, POST
  • trading -> /graph/index/61, POST
  • others -> /graph/index/53, POST

Company Information

  • List -> /company/list
  • Classification -> /security/classification
  • Graph Data -> /market/graphdata/131, POST
  • Detail -> /security/8045
  • price_volume_history -> /market/security/price/8045
  • Stock Trading -> /market/history/security/2790?&size=500&startDate=2021-08-23&endDate=2022-08-23
  • floorsheet -> /security/floorsheet/8045, POST

Market Statistics

  • Market Summary -> /market-summary
  • Market History -> /market-summary-history
  • Market Capitilization by Date -> /nepse-data/marcapbydate/
  • Supply Demand -> /nepse-data/supplydemand
  • Top Gainers -> /top-ten/top-gainer?all=true
  • Top Losers -> /top-ten/top-loser?all=true
  • Top Turnover -> /top-ten/turnover?all=true
  • Top Volume -> /top-ten/trade?all=true
  • Top Transaction -> /top-ten/transaction?all=true
  • Nepse Index -> /nepse-index
  • FloorSheet -> /nepse-data/floorsheet?page=0&size=500&sort=contractId,desc, POST
  • Indexes -> /index
  • Indices -> /index/history/58?
  • N days trading average price -> /nepse-data/trading-average?nDays=120
  • Today's Price -> /nepse-data/today-price, POST
  • Sectorwise Options -> /sectorwise
  • Sectorwise Details -> /sectorwise?businessDate=2022-08-08

Broker Information

  • Broker List -> /member?&size=500, POST
    {"memberName":"","contactPerson":"","contactNumber":"","memberCode":"","provinceId":0,"districtId":0,"municipalityId":0,"isDealer":"Y"}
  • Stock Dealers -> /member/dealer?&size=500, POST
    {"memberName":"","contactPerson":"","contactNumber":"","memberCode":"","provinceId":0,"districtId":0,"municipalityId":0,"isDealer":"Y"}

Other Information

  • Promoter Share -> /security/promoters
  • Holyday Options -> /holiday/year
  • Holiday by Year -> /holiday/list?year=2022
